# 体育祭 順番待ちシステム

体育祭の各種目（**ストラックアウト** / **ジャイアントボーリング**）の順番待ち状況を、Google フォーム + スプレッドシートだけで管理する軽量システムです。

## 主な機能

- 📱 **QRコードで受付** — 参加者がスマホでフォーム送信すると自動でキューに追加
- 🖥️ **大型表示画面** — 呼出中の番号 / 待ち人数 / 推定待ち時間 / 次の人を表示
- 🎛️ **運営管理画面** — 「呼出」「完了」「取消」ボタンで運用
- 🖨️ **QR印刷ページ** — フォームURLから印刷用QRを生成
- 🏷️ **2種目の同時管理** — 種目ごとに完全分離して表示

## クイックスタート

詳細手順は **[SETUP.md](./SETUP.md)** を参照してください。

```
1. Google フォーム を作成 (氏名・学籍番号・種目)
2. 自動生成されるスプレッドシートに Apps Script を設定 (apps-script/Code.gs)
3. デプロイした Web App URL を config.js の apiUrl に貼る
4. index.html / admin.html / qr.html をブラウザで開く
```

## ファイル

| ファイル | 用途 |
|----------|------|
| `index.html` | `view.html` へ自動リダイレクト |
| `view.html` | 受付者がスマホで待ち状況を確認するページ |
| `monitor.html` | 大型モニタ表示（呼出中・待ち人数） |
| `admin.html` | 運営向け管理画面（呼出・完了操作） |
| `qr.html` | 受付用QRコードの生成と印刷 |
| `config.js` | API URL や種目設定 |
| `app.js` / `styles.css` | 共通ロジック・スタイル |
| `apps-script/Code.gs` | スプレッドシート連携APIのコード |
| `SETUP.md` | 詳細セットアップ手順 |
